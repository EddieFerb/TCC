Central de Ajuda

Bem-vindo! Aqui estão as principais orientações de uso do Dashboard.

Contraste e Tema

Use os botões no topo para alternar entre tema claro, escuro e contraste invertido.

Ajuste de Fonte

Use os botões A+ e A− para ajustar o tamanho da fonte até 200%.

Navegação por Teclado

  • Tab para avançar pelos elementos interativos.
  • Shift + Tab para voltar.

Atalho Ação sugerida

ALT + AFocar no Botão Ajuda Contextual (se diferente da página de Ajuda principal)
ALT + KAtivar teclado virtual na tela (para quem tem mobilidade reduzida)
ALT + DAlternar para Modo de Alto Contraste para Daltonismo (se for um modo específico)
ALT + PAlternar Ponteiro de Mouse Ampliado
CTRL + ALT + UAtivar/desativar Legenda Automática (caso tenha vídeos)
ALT + TIr direto para tutorial de acessibilidade (fluxo guiado explicando como usar os recursos)
